# VisualQ Pilot

## Problem

CBSE question paper generation tools can generate questions but cannot intelligently determine when a visual diagram is required and generate that diagram automatically.

## Solution

VisualQ Pilot is an AI-powered educational assessment platform that generates CBSE-aligned unit tests using NCERT textbooks and CBSE PYQs while automatically detecting, generating, and embedding relevant diagrams.

## Users

- Teachers
- Coaching Institutes
- Students
- Educational Organizations

## Core Features

- CBSE Unit Test Generation
- RAG-based Question Generation
- Diagram Detection
- Free Body Diagram Generation
- Circuit Diagram Generation (Bonus)
- Mathematical Graph Generation (Bonus)
- PDF Export

## Success Metrics

- Accurate CBSE-style questions
- Relevant diagram generation
- Low latency generation
- High retrieval relevance
