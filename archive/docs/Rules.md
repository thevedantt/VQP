# Development Rules

## Backend

- FastAPI only
- Modular architecture
- Service-based structure

## Frontend

- Next.js App Router
- TypeScript
- shadcn/ui

## Database

- PostgreSQL
- pgvector

## AI

- Structured JSON outputs
- No free-text agent communication

## Code Quality

- Typed APIs
- Clear naming
- Reusable components
- Logging enabled

## Architecture Principle

Every feature must be explainable in one sentence.
